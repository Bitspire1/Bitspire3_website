## About the Project

Bezawaryjni is a professional landing page for a plumbing company. The project focuses on a simple, clear message and lightning-fast loading speed, which is crucial for local businesses.

## Challenges

- Creating a minimalist yet effective design
- Optimization for mobile devices (main traffic source)
- Maximizing performance and SEO for local searches
- Clear presentation of services and contact information

## Technical Solutions

The website was built with Next.js 14 using:
- **Static Site Generation** for maximum performance
- **Responsive Design** with mobile-first approach
- **Image optimization** in WebP/AVIF format
- **SEO** optimized for local searches
- **Core Web Vitals** at the highest level

## Results

The project achieved:
- **100/100** in Google PageSpeed Insights
- Loading time under 1 second
- Full responsiveness on all devices
- High position in local search results

## Technologies

- Next.js 14
- TypeScript
- TailwindCSS
- Vercel (hosting)
