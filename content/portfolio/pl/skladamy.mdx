## O projekcie

SkładaMy to nowoczesna platforma e-commerce specjalizująca się w sprzedaży mebli modułowych. Kluczowym elementem projektu był zaawansowany konfigurator produktów, który pozwala użytkownikom na personalizację mebli w czasie rzeczywistym.

## Wyzwania

- Stworzenie intuicyjnego interfejsu konfiguratora z podglądem w czasie rzeczywistym
- Optymalizacja wydajności przy złożonych interakcjach użytkownika
- Integracja z systemem zarządzania magazynem (API)
- Osiągnięcie wysokich wyników Core Web Vitals
- Responsywność na wszystkich urządzeniach

## Rozwiązanie

### Architektura techniczna

Zdecydowaliśmy się na **Next.js 14** z App Router, co pozwoliło nam na:
- Server-side rendering dla lepszego SEO
- Optymalne ładowanie zasobów
- Wykorzystanie React Server Components dla lepszej wydajności

### Konfigurator produktów

Interaktywny konfigurator został zbudowany z wykorzystaniem:
- **React State Management** – wydajna obsługa stanu konfiguracji
- **Canvas API** – renderowanie podglądu produktu w czasie rzeczywistym
- **Debouncing** – optymalizacja wywołań API podczas zmian konfiguracji

### Optymalizacja wydajności

- **Image Optimization** – automatyczna optymalizacja obrazów przez Next.js
- **Code Splitting** – podział kodu na mniejsze chunki
- **Lazy Loading** – opóźnione ładowanie komponentów
- **Edge Caching** – wykorzystanie CDN dla statycznych zasobów

## Wyniki

- ✅ **Core Web Vitals**: wszystkie metryki w zielonym zakresie
- ✅ **LCP** (Largest Contentful Paint): < 2.5s
- ✅ **FID** (First Input Delay): < 100ms
- ✅ **CLS** (Cumulative Layout Shift): < 0.1
- ✅ **Wzrost konwersji**: +35% w porównaniu do poprzedniej wersji
- ✅ **Zwiększenie czasu sesji**: +45%

## Technologie

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- React Hook Form
- Zustand (state management)
- REST API integration
- Vercel (hosting)
